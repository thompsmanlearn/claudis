## Key n8n Build Bug Learned (permanent)
- n8n HTTP Request with pre-serialized JSON body: requires `contentType: raw` (NOT `specifyBody: "raw"`)
- Without `contentType: raw`, n8n double-encodes the string → Anthropic receives a string, not an object
- Pattern: match the 4-Pillars evaluator's "Call Sonnet Evaluator" node exactly
- Sandbox Notify needs `onError: continueRegularOutput` or it blocks the webhook response
