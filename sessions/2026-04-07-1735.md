## Decisions

- Audit body must be pre-serialized in the upstream Code node — never call `JSON.stringify()` inside n8n expression syntax.
- `Write Audit Log` branches from `Write to Supabase` (parallel with `Send to Bill`) — not from `Format Output`. This keeps the chain semantically correct: audit confirms the write happened, not just that output was formatted.
- serendipity_engine_prod has `triggerCount: 2` (Daily Schedule + Test Webhook). This is NOT a V2 violation — the Test Webhook is manual-only, not scheduled. No double-fire risk. No action needed.
