### 2. autonomous_growth_scheduler — Diagnosed and Fixed Silent Failure

**Discovery:** The autonomous_growth_scheduler (Lm68vpmIyLfeFawa) has been active in n8n since April 6 but was failing on every 6-hour tick. Checked execution history: 5/5 recent executions all `error` status.

**Root cause:** Missing `sendBody: true` flag on the "Insert Work Item" HTTP Request node. Without it, n8n sends an empty POST body to Supabase even though the body expression is configured. Supabase returns "400 Bad request — Empty or invalid json". The Gate, rotation lookup, and payload build all passed correctly — only the INSERT was broken.

**Impact:** Every sentinel session since April 6 arrived to an empty work queue because the scheduler never successfully inserted a task. 10+ days of autonomous growth silently aborted at the last step.

**Fix:** Added `sendBody: true` to both "Insert Work Item" and "Update Rotation" nodes. Updated agent_registry status from `building` → `active`.

**Verification:** Cannot verify until next scheduled run at ~07:00 UTC. Noted in handoff.
