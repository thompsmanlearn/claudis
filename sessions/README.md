# /sessions

One artifact per session. Written at session close, after all GitHub commits exist so narratives can reference them.

**Naming:** `YYYY-MM-DD-HHMM.md`

**Contents:** Date, tasks completed, lessons written, branches opened/closed, key decisions, pointers to commits. Metadata structured, narrative body free-form.

See `/sessions/monthly/` for end-of-month development snapshots written with external readability as primary goal.
