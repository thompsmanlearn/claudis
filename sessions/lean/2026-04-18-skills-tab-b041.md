# Session: 2026-04-18 — Skills tab (B-041)

## Directive
B-041: Add a Skills tab to the Anvil dashboard showing all registered skills with content drill-down.

## What Changed

- **uplink_server.py**: Added `get_skills()` (returns all registry rows) and `get_skill(name)` (reads file content from filesystem on demand). `aadp-anvil.service` restarted.

## What Was Learned
- Skills tab follows exact same lazy-load pattern as Lessons and Memory tabs: `_skills_loaded` flag, load on first visit.
- `file_path` stored with `~` prefix; `get_skill()` expands it with `os.path.expanduser` — clean and portable.
- No `times_loaded` increment wired in yet (deferred per B-041 card scope — would require boot chain change).
